# Costs

## Estimated Monthly Cost

* EC2 (3 × t3.micro): $25–30
* Application Load Balancer: $18–20
* NAT Gateway: $30–35

Estimated total: $70–90/month

---

## Cost Optimisation

* Use of small instance types (t3.micro)
* Infrastructure destroyed when not in use
* Fixed scaling to avoid unexpected cost spikes
* Efficient use of load balancing instead of over-provisioning

---

## Trade-offs

* Lower instance size reduces cost but limits performance
* NAT Gateway provides security but increases cost
* Fixed scaling improves predictability but reduces flexibility

---

## Notes

Costs vary based on usage and region.
